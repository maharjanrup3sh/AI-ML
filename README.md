# 🌍 Analyzing Temporal Patterns of Earthquakes in Nepal (2015–2025)

A data analysis and machine learning project that explores earthquake activity in Nepal over a 10-year period using real seismic data.

---

## 📌 Project Overview

This project analyzes earthquake data from Nepal between 2015 and 2025 to identify temporal patterns, magnitude distributions, and geographic clusters of seismic activity. It combines data analysis, visualization, and unsupervised machine learning (KMeans clustering) to draw meaningful insights from the data.

---

## 🛠️ Tools & Libraries Used

- **Python**
- **Pandas** — data loading and preprocessing
- **Matplotlib & Seaborn** — data visualization
- **Scikit-learn** — KMeans clustering, StandardScaler
- **Folium** — interactive map visualization
- **GeoPandas** — geographic data handling

---

## 📊 Key Analysis Performed

- **Yearly Earthquake Frequency** — bar chart showing number of earthquakes per year
- **Magnitude Distribution** — histogram with KDE to understand earthquake intensity spread
- **Epicenter Mapping** — scatter plot of earthquake locations colored by magnitude
- **Average Magnitude per Year** — trend line showing magnitude changes over time
- **KMeans Spatial Clustering** — grouped earthquake hotspots into 3 geographic clusters
- **Interactive Map** — Folium-based HTML map showing all earthquake clusters across Nepal

---

## 🗂️ Dataset

- **File:** `nepal_earthquakes_2015_2025.csv`
- **Columns used:** Time, Latitude, Longitude, Magnitude
- **Period:** 2015 – 2025

---

## 🚀 How to Run

1. Clone the repository
2. Upload the dataset `nepal_earthquakes_2015_2025.csv`
3. Open the notebook `Analyze_Temporal_Patterns_Earthquake_in_Nepal.ipynb`
4. Run all cells in order
