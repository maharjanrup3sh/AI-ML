AI Pipeline Assistant for Blender

-------------------------------------------------
DESCRIPTION
-------------------------------------------------
AI Pipeline Assistant v4 is a Blender addon that helps you automatically organize, classify, rename, and optimize your 3D scenes.

It is designed for game developers, environment artists, and production workflows.

-------------------------------------------------
FEATURES
-------------------------------------------------
- Automatic object classification (ENV / PROP / CHAR)
- Smart naming system (Game / Film / Simple modes)
- Auto collection organization
- Scene optimization (decimate, texture scaling, duplicate cleanup)
- Simple AI-style learning memory system

-------------------------------------------------
INSTALLATION
-------------------------------------------------
1. Open Blender
2. Go to Edit → Preferences → Add-ons
3. Click "Install"
4. Select the downloaded ZIP file
5. Enable "AI Pipeline Assistant v4"
6. Open 3D Viewport → Press N → Pipeline tab

-------------------------------------------------
USAGE
-------------------------------------------------
1. Select objects in your scene
2. Click "Scan Scene" to analyze
3. Click "Apply Naming & Grouping" to organize
4. Click "Optimize Scene" to improve performance

-------------------------------------------------
REQUIREMENTS
-------------------------------------------------
- Blender 3.0 or higher

-------------------------------------------------
NOTES
-------------------------------------------------
- Works on selected objects
- Optimization is non-destructive (uses modifiers where possible)
- Best used in production or large scenes
