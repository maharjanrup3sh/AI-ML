# 📚 Deep Learning with FastAI

A collection of practice notebooks and exercises based on the book **"Deep Learning for Coders with FastAI and PyTorch"** by Jeremy Howard & Sylvain Gugger.

---

## 📌 About This Repository

All files in this repository are personal practice work following along with the FastAI book. The notebooks cover hands-on deep learning concepts, experiments, and exercises as taught in the book — from beginner fundamentals to advanced deep learning techniques.

---

## 📖 Based On

**Book:** Deep Learning for Coders with FastAI and PyTorch
**Authors:** Jeremy Howard & Sylvain Gugger
**Publisher:** O'Reilly Media
**Official Course:** [fast.ai](https://www.fast.ai)

---

## 🛠️ Tools & Libraries Used

- **Python**
- **FastAI** — high-level deep learning library
- **PyTorch** — underlying deep learning framework
- **Jupyter Notebook / Google Colab** — coding environment

---

## 📂 What's Inside

All notebooks are personal practice files covering topics such as:

- Image classification
- Data augmentation and preprocessing
- Transfer learning
- Training and fine-tuning models
- Deep learning fundamentals

> ⚠️ **Note:** These files are for personal learning and practice purposes only. All concepts, methods, and techniques are based on and credited to the FastAI book and course.

---

## 🚀 How to Run

1. Clone the repository
2. Open notebooks in **Jupyter Notebook** or **Google Colab**
3. Install FastAI: `pip install fastai`
4. Run cells in order
